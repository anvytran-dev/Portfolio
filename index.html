Recent Work

The Sign Up Cube: A Volunteer Organizing App

A full stack application that helps you organize your volunteers by making the sign-up process easy! Just create a sign-up sheet and watch your volunteers sign up!

Resilient Roasters: A Barista App

This is a full stack web application for 'Resilient Roasters', a speciality coffee shop. The cashier can take the customer's order and send it to the baristas to fulfill. The baristas can create accounts and login to see which drink orders are pending. When they complete the order, baristas can put it in the completed queue.

Wallet Watcher: A Spending Tracker App

This is a full-stack appliaction where the user can track their spending. They can detail what they purchased and how much it was. The user can edit the purchase notes that they have created. They can check how much they spent on a certain day!

Drive-In: Movie Recommender App

This is a movie recommender app that uses 2 APIs. It takes an actor's name and generates a movie recommendation and the movie review. For example, if the user wants to watch a movie with Will Smith, they can input Will Smith and click enter. Then, a movie recommendation with Will Smith in it and a movie review will appear.

Forest Tic-Tac-Toe: OOP

A blissful afternoon in the forest sets this game of tic-tac-toe which is created using object-oriented programming.

Harry Potter Creatures: Matching Game

This is a Harry Potter-themed, matching game application. The player must match the cards with the same image together. There are 6 pairs of images, so there are 6 matches. The cards are shuffled using the Fisher-Yates Shuffle.
